## Local Setup and Run Steps

### Prerequisites

- [Node.js LTS](https://nodejs.org/) (includes npm)

### 1) Install dependencies

```bash
cd taskflow
npm install
```

If PowerShell cannot find npm, install Node.js first, reopen terminal, then run:

```bash
node -v
npm -v
```

### 2) Run REST API (JSON Server)

```bash
npm run api
```

API endpoint:

```text
http://localhost:3000/tasks
```

### 3) Run frontend on a local server

Open a second terminal in the same folder:

```bash
npm run serve
```

Open:

```text
http://localhost:5500
```

Do not open `index.html` directly by double-clicking. Run with a local server so `fetch()` works correctly with the API.

## REST API Contract (Act 2)

| Method | Endpoint     | Purpose |
|--------|--------------|---------|
| GET    | `/tasks`     | Read all tasks |
| POST   | `/tasks`     | Create task |
| PUT    | `/tasks/:id` | Update task |
| DELETE | `/tasks/:id` | Delete task |

## Task Data Model

| Field | Type | Notes |
|-------|------|-------|
| id | number/string | Generated by server |
| title | string | Required |
| description | string | Optional |
| priority | string | `low`, `medium`, `high` |
| status | string | `pending`, `in-progress`, `done` |
| dueDate | string | ISO date (`YYYY-MM-DD`) |
